## Perché Meteora System?

Hai mai avuto la sensazione di avere troppi plugin da gestire, con interfacce diverse e logiche che a volte si pestano i piedi? Meteora System nasce proprio da questa esigenza: **unificare** in un'unica piattaforma modulare tutti gli strumenti essenziali per chi fa e-commerce e content marketing su WordPress.

- **Coerenza Totale:** Un'unica interfaccia, un unico menu, una logica condivisa.
- **Performance Ottimizzata:** Niente più plugin duplicati o in conflitto. Il core condiviso gestisce API, logging e database in modo efficiente.
- **Flessibilità Estrema:** Ogni modulo può essere attivato o disattivato singolarmente. Vuoi solo il sistema fidelity? Attivi solo quello.
- **Pronto per il Futuro:** Integra le più recenti tecnologie WordPress, incluse le **Abilities API** (AI-Ready), React per interfacce dinamiche e REST API personalizzate.

## ✨ Panoramica delle Funzionalità

### 1. SEO Automation & Ottimizzazione AI (Meteora SEO)
- **Ottimizzazione Massiva:** Analizza e ottimizza automaticamente titoli, meta description e contenuti di articoli, pagine, prodotti WooCommerce, categorie, tag e brand.
- **Motore AI Gemini:** Utilizza le API di Google Gemini per suggerimenti SEO contestuali e miglioramenti in linguaggio naturale.
- **Sistema di Rollback:** Hai fatto un errore? Con un click puoi ripristinare lo stato SEO precedente a qualsiasi modifica.

### 2. Gestione Saldi Massivi per WooCommerce
- **Applica/Rimuovi Saldi in Massa:** Seleziona prodotti per categoria, tag o specifici SKU e applica una percentuale di sconto (o un prezzo fisso) in pochi secondi.
- **Programmazione Intelligente:** Imposta date di inizio e fine saldi; il plugin gestirà automaticamente l'attivazione e la rimozione.
- **Tab Dedicata:** Pannello di controllo specifico per visualizzare tutti i saldi attivi e gestirli.

### 3. Generazione Automatica di Articoli con Gemini AI
- **Alimentato da Feed RSS:** Inserisci uno o più feed RSS di settore. Il plugin li analizza periodicamente.
- **Selezione Intelligente:** Gemini AI valuta la qualità e la pertinenza degli articoli, selezionando solo i migliori da riscrivere.
- **Creazione Post Completi:** Genera articoli unici (numero a scelta) con titolo, contenuto, excerpt e categorie suggerite. Pronti per la revisione o la pubblicazione automatica.

### 4. Recupero Carrelli Abbandonati
- **Banner Interattivo:** Un banner discreto ma efficace nella pagina del carrello invita l'utente a lasciare la propria email in cambio di un promemoria.
- **Memorizzazione Dettagliata:** Il plugin salva il contenuto del carrello (prodotti, quantità, valori) associato all'email.
- **Invio Promemoria:** Puoi inviare manualmente o automaticamente un'email di promemoria personalizzata, aumentando le probabilità di conversione.

### 5. Sistema Fidelity, Coupon e Ticketing (ex Unique Coupon Generator)
Questa è la parte più corposa e potente della suite, un vero e proprio CRM di marketing.

- **Generazione Coupon Unici con QR Code:**
  - Crea set di coupon illimitati con campi personalizzati, immagini dedicate e stato (bozza/attivo/chiuso).
  - Ogni coupon generato per un utente include un QR code univoco e tracciabile.
  - Consegna multipla: download immediato in PNG/PDF o invio via WhatsApp con link diretto.

- **Programma Fidelity Integrato:**
  - Accumulo punti per ogni euro speso (configurabile).
  - Bonus di iscrizione.
  - Log dettagliati delle transazioni e saldo punti visibile in back-office e front-end.

- **Gestione Eventi e Ticketing:**
  - Crea eventi con date, luoghi, tipologie di ticket e blocchi di vendita.
  - Wizard front-end a step per l'acquisto (raccolta dati, scelta ticket, pagamento).
  - Pagamenti in loco (manuale) o integrazione nativa con WooCommerce.
  - Dopo il pagamento, l'utente viene reindirizzato per scaricare il ticket o riceverlo via WhatsApp.

- **Check-in e Pannello PR:**
  - Scanner QR code HTML5 per verificare i ticket all'ingresso.
  - Pannello riservato ai PR per gestire pagamenti, vedere informazioni clienti e esportare report.

- **Marketing e CRM:**
  - Anagrafiche clienti unificate (dati da coupon ed eventi).
  - Segmentazione per età, città, tipologia di coupon o evento acquistato.
  - Invio campagne email con template personalizzabili e tracking degli esiti.

### 6. Strumenti di Diagnostica e Performance
- **Analisi Server vs WordPress:** Un pannello che monitora i tempi di risposta e ti aiuta a capire se un rallentamento dipende dal server host o da qualche plugin/theme.
- **Log Errori Centralizzato:** Tutti gli errori dei moduli vengono registrati in una tabella dedicata (`wp_meteora_error_log`) e in un file di log, consultabili da una comoda pagina admin.
- **Strumenti Vitali:** Pulizia cache, controllo integrità database, informazioni sul sistema in tempo reale.

## ⚙️ Tecnologie e Standard Moderni Integrati

Questo plugin non è solo "codice che funziona", è un esempio di **ingegneria del software moderna** applicata a WordPress.

- **Namespace e Autoloading PSR-4:** Tutte le classi sono organizzate in namespace (es. `Meteora\Seo\Analyzer`) e caricate automaticamente con Composer. Zero conflitti, zero `require_once` sparsi.
- **React per UI Complesse:** I pannelli di amministrazione più complessi (es. gestione fidelity, reportistica CRM) sono costruiti con React e TypeScript, offrendo un'esperienza utente fluida e reattiva. Il build è gestito con `@wordpress/scripts`.
- **REST API Personalizzate:** Ogni modulo espone i propri dati tramite REST API, permettendo al front-end React di comunicare in modo efficiente e sicuro con il back-end.
- **Abilities API (WordPress 6.9+):** Meteora System è **AI-Ready**. Registra le proprie "abilità" (es. "genera coupon", "ottimizza SEO") seguendo il nuovo standard WordPress. Questo permette ad assistenti AI (come Jules) di interagire programmaticamente con il plugin in modo strutturato, aprendo scenari di automazione futuribili.
- **Sviluppo Basato su Standard:** Il codice segue le linee guida di codifica di WordPress, verificato automaticamente da PHP_CodeSniffer nei workflow GitHub.
- **Database Robusto:** Utilizzo di tabelle personalizzate ben strutturate, con chiavi e indici per performance ottimali. Ogni modulo gestisce le proprie tabelle, ma il core fornisce helper per la creazione e l'upgrade.

## 📦 Requisiti di Sistema

- **WordPress:** 5.0 o superiore (consigliato 6.0+ per sfruttare appieno React e Abilities API).
- **PHP:** 7.4 o superiore (PHP 8.0+ fortemente consigliato per performance e sicurezza).
- **WooCommerce:** 5.0 o superiore (obbligatorio solo se si attivano i moduli `saldi`, `carrelli-persi` o la parte di ticketing con pagamento WooCommerce).
- **Database:** MySQL 5.6+ o MariaDB 10.1+.
- **Memoria PHP:** Almeno 128MB dedicati a WordPress (256MB consigliati).

## 🚀 Installazione e Primi Passi

### Installazione Rapida (ZIP)
1. Vai alla sezione [Releases](link-alla-release) di questo repository.
2. Scarica l'ultimo file `meteora-system.zip`.
3. Accedi al pannello di amministrazione di WordPress, vai su **Plugin > Aggiungi nuovo > Carica plugin**.
4. Scegli il file ZIP e clicca "Installa ora".
5. Attiva il plugin.

### Sviluppo e Test Istantanei con WordPress Playground (Consigliato!)
Non vuoi installare nulla sul tuo computer? Vuoi vedere subito il plugin in azione o sviluppare una funzionalità in un attimo? Usa WordPress Playground direttamente dal browser.

[![Apri in WordPress Playground](https://img.shields.io/badge/Apri_in-WordPress_Playground-21759B?style=for-the-badge&logo=wordpress)](https://playground.wordpress.net/?plugin=IL_TUO_USERNAME_GITHUB/meteora-system)

**Sostituisci `IL_TUO_USERNAME_GITHUB` con il tuo username GitHub.**  
Verrà avviata un'istanza WordPress fresca e volatile con Meteora System già installato e attivo. Perfetto per test rapidi, demo o sviluppo senza pensieri.

### Ambiente Locale Avanzato (Opzionale)
Se preferisci un ambiente locale per uno sviluppo più profondo (con debugger, XDebug, ecc.), puoi usare `wp-env` (richiede Node.js e Docker):

```bash
# Clona la repository
git clone https://github.com/IL_TUO_USERNAME_GITHUB/meteora-system.git
cd meteora-system

# Installa le dipendenze
npm install
composer install

# Avvia l'ambiente WordPress locale
npm run env:start

# Ora visita http://localhost:8888 (wp-admin: admin / password)
```

## 🧪 Sviluppo, Testing e Quality Assurance

Mantenere alta la qualità del codice è fondamentale. La repo è configurata con una serie di strumenti automatici.

- **Linting PHP:** Controlla lo stile del codice seguendo gli standard WordPress.
  ```bash
  composer lint
  ```
- **Linting JavaScript:** Controlla lo stile del codice JS/React.
  ```bash
  npm run lint:js
  ```
- **Test PHPUnit:** Esegue la suite di test per verificare che le funzioni core e dei moduli lavorino correttamente.
  ```bash
  composer test
  ```
- **Build Assets:** Compila i file React/JavaScript e CSS per la produzione.
  ```bash
  npm run build
  ```
- **GitHub Actions:** Ad ogni push, vengono eseguiti automaticamente linting e test. Ad ogni release, viene generato automaticamente il file ZIP del plugin.
