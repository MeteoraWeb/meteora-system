# Traduzioni del plugin

Questa cartella contiene i file necessari per la localizzazione del plugin **Unique Coupon Generator**.

- `unique-coupon-generator.pot`: template di riferimento generato con `wp i18n make-pot`.
- `unique-coupon-generator-it_IT.po`: traduzione italiana basata sul template.
- `unique-coupon-generator-it_IT.mo`: generare questo file binario a partire dal `.po` (non incluso nel repository) per l'uso in produzione.

Aggiungi qui eventuali altre traduzioni seguendo la convenzione `unique-coupon-generator-LL_CC.po` / `.mo`.
