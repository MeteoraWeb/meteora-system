----------------------------------------------------
INSTALLATION
----------------------------------------------------
1. Log into your WordPress dashboard.
2. Go to Plugins → Add New → Upload Plugin.
3. Select the file "installable-plugin.zip".
4. Click "Install Now" and then "Activate".
5. A new menu item "Coupon & Fidelity" will appear in your admin sidebar.
